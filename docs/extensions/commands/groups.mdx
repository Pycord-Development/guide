title: Command Groups
---

Command groups are a way to create subcommands for your commands. For example, `!afk set` or `!afk
remove`.

## Prefixed Command Groups

Creating a command group is very simple.

```python title="Command Groups Example"
import discord
from discord.ext import commands

bot = commands.Bot(command_prefix='!')

@bot.group()
async def afk():
    pass

@afk.command()
async def set():
    ...

@afk.command()
async def remove():
    ...

# Another command group

@bot.group(invoke_without_command=True) # Indicates if the group callback should begin parsing and invocation only if no subcommand was found.
async def math(ctx):
    await ctx.send('Subcommand not found')

@math.command()
async def add(ctx, a: int, b: int):
    ...

@math.command()
async def subtract(ctx, a: int, b: int):
    ...
```

To create a command group, the command's decorator must be `@bot.group`. Once you have a command with
that decorator, you can use `@function.command()`, such as `@math.command()`. Now, you have subcommand
groups!


## Slash Command Groups

Slash command groups are a bit more complicated. You must create a `SlashCommandGroup` object, and then
add subcommands to it. You can then add the group to the bot.

```python title="Slash Command Groups Example"
import discord
from discord.ext import commands
from discord import SlashCommandGroup

bot = commands.Bot(debug_guilds=[...])

# Automatically create a Slash Command Group
math = bot.create_group(
    "math", "Commands related to mathematics."
)

# The manual way, creating a SlashCommandGroup object and adding it to the bot.
math = discord.SlashCommandGroup("math", "Commands related to mathematics.")

# Create a sub-group
advanced = math.create_subgroup("advanced", "Advanced mathematics commands.")

# /math add
@math.command()
async def add(ctx, a: int, b: int):
    await ctx.respond(f"The sub of these numbers is {a + b}")

/math subtract
@math.command()
async def subtract(ctx, a: int, b: int):
    await ctx.respond(f"The sub of these numbers is {a - b}")

/math advanced multiply
@advanced.command()
async def multiply(ctx, a: int, b: int):
    await ctx.respond(f"The product of these numbers is {a * b}")

/math advanced divide
@advanced.command()
async def divide(ctx, a: int, b: int):
    await ctx.respond(f"The quotient of these numbers is {a / b}")

# If you created the group manually, you must add it to the bot.
bot.add_slash_command_group(math)
bot.add_slash_command_group(advanced)


bot.run("token")
```
